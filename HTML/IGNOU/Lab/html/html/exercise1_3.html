Basics of HTML

Web pages or material in the form of hypermedia documents accessed through the Internet can be located anywhere in the world.

No matter where they originated, most Web documents are creatd using Hypertext Markup Language (HTML). HTML is a powerful authoring language and has different versions like HTML 4.2, HTML 4.0, HTML 3.2 and HTML 2.

HTML is used to define document structure and format, with the help of a single tag or a pair of tags. A tag is a string in the language sourrounded by a less than (<) and a greater than (>) sign. An opening tag does not begin with a slash (/). And ending or closing tag is a string that begins with a slash (/).
